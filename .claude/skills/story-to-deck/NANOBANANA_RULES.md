# NANOBANANA_RULES.md — nanobanana2プロンプト生成ルール

nanobanana2（AIイメージ生成ツール）でスライド用画像を生成するためのプロンプトを設計し、MDの所定フィールドに記載するルール。

---

## Step 1: nanobanana2使用確認

**解析を始める前に必ず**ユーザーに確認する:

> 「nanobanana2による生成画像の挿入プランにしますか？  
> Yesの場合、plain系スライドに画像プレースホルダーとnanobanana2用プロンプトを追加します。  
> 生成した画像を後で手動で貼り付けることができます。」

---

## ルールA: image_label への画像プロンプト

**対象テンプレート:** `plain_image_row`, `plain_image_col`

**書き込み先:** フロントマターテーブルの `image_label_1` フィールド

### image_row（16:5 横長ワイド）のプロンプト形式

```
[スライドテーマ]の概念を示す横長インフォグラフィック。フラットデザイン、ミニマル、ビジネス向け、白背景。16:5の横長比率で[要素1]・[要素2]・[要素3]を視覚的に図解する。
```

**使用シーン例:**
- 全体フレームワーク・体系を俯瞰するスライド
- タイムライン・変遷を示すスライド
- 比較・フロー・関係性を図示するスライド

### image_col（6:5 ほぼ正方形）のプロンプト形式

```
[スライドテーマ]を表すシンプルなイラスト。フラットデザイン、ミニマルアイコン風、ビジネス向け、白背景。6:5の比率で[具体的な概念]を1枚のビジュアルで表現する。
```

**使用シーン例:**
- 単一コンセプトを視覚化するスライド
- テキスト説明を補完するイラストが欲しいスライド
- 1カラムのテキストに対してサイドに図を置くスライド

## ルールB: note へのアイコン一括生成プロンプト

**対象テンプレート:** `list_3card`, `flow_3step`, `flow_4step`, `diffuse_3card`, `converge_3card`, `bg_3card`

**書き込み先:** `note` フィールドの末尾に追記（発表者ノートとして保存され、PPTXで参照できる）

**方針:** 1回の生成で全アイコンを横一列に並べた1枚の画像を出力し、ユーザーが手動でトリミングして各カード/ステップのアイコンとして使用できるようにする。

### プロンプト形式

```
[nanobanana2 icon prompt]
[N]つのフラットアイコンを横一列に並べた1枚の画像を生成する。
左から順に:
  アイコン1: [カードAのテーマを表す概念・シンボル]
  アイコン2: [カードBのテーマを表す概念・シンボル]
  アイコン3: [カードCのテーマを表す概念・シンボル]
各アイコンは正方形の枠内に収まるシンプルなフラットデザイン。白背景、色統一、ミニマルスタイル。
※生成後に水平[N]等分にトリミングして各カード/ステップのアイコンとして使用。
```

### アイコン数・トリミング方法

| テンプレート | アイコン数 | トリミング |
|------------|----------|---------|
| `list_3card` | 3 | 水平3等分 |
| `flow_3step` | 3 | 水平3等分 |
| `flow_4step` | 4 | 水平4等分 |
| `diffuse_3card` | 3 | 水平3等分 |
| `converge_3card` | 3 | 水平3等分 |
| `bg_3card` | 3 | 水平3等分 |

## プロンプト品質ガイドライン

nanobanana2プロンプトを生成する際は以下を意識する:

1. **具体性**: 「ビジネス」ではなく「会議室でのプレゼン」など具体的なシンボル・シーンを指定する
2. **スタイル統一**: 全スライドで「フラットデザイン、白背景、ミニマル」を一貫させる
3. **日本語対応**: プロンプトは日本語で書いてよい（nanobanana2は日本語に対応している）
4. **アスペクト比の明示**: image_rowは「16:5の横長比率」、image_colは「6:5の比率」を必ず明記する
5. **テキスト不要**: 画像内にテキスト・文字を含めないよう指定する（「文字なし」を追記するとよい）

## 記載例

### plain_image_row の例

```markdown
| key | value |
|-----|-------|
| layout | plain_image_row |
| image_label_1 | AIの活用により開発生産性が向上するプロセスを示す横長インフォグラフィック。フラットデザイン、白背景、文字なし。16:5の横長比率で「データ収集→モデル学習→自動化実装」の3ステップを矢印でつないで図解する。 |
| note | 発表者ノート：AI導入の3フェーズを順を追って説明する。各フェーズのKPIを補足すること。 |
```

### list_3card のnote例

```markdown
| key | value |
|-----|-------|
| layout | list_3card |
| note | 発表者ノート：3つのメリットをそれぞれ30秒で説明する。[nanobanana2 icon prompt] 3つのフラットアイコンを横一列に並べた1枚の画像を生成する。左から順に: アイコン1: 速度・時計のシンボル アイコン2: コスト削減・グラフ下向きのシンボル アイコン3: 品質向上・星・チェックマークのシンボル。各アイコンは正方形の枠内に収まるシンプルなフラットデザイン。白背景、青系色統一、ミニマルスタイル、文字なし。※生成後に水平3等分にトリミングして各カードのアイコンとして使用。 |
```
